1- ¿Para qué sirve la etiqueta ul? Dar un ejemplo en donde lo utilizarías.

Ul sirve para crear una lista de forma desordenada, un ejemplo podria ser:
Utiles

  • Goma
  • Plasticola
  • Tijera

2- ¿Para qué sirve la etiqueta ol? Dar un ejemplo en donde lo utilizarías.

Ol sirve para crear una lista de forma ordenada, es decir, siguen un orden, ya sea numerico o alfabetico

3- ¿Para qué sirve la etiqueta dl? Dar un ejemplo en donde lo utilizarías.

Dl sirve para crear una lista en la que se guardan elementos y una breve descripcion de cada uno, mientras que el elemento se marca con dt y la descripcion se mrca con dd

4- ¿Para qué sirve la etiqueta li? Dar un ejemplo en donde lo utilizarías

Li sirve para separar los elementos de una lista ya especificada, es decir, siempre va acompañado de un ul o un ol
